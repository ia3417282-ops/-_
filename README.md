# نظام أعمالي - Aamali ERP

نظام متكامل لإدارة الأعمال والموارد البشرية والمبيعات والمخزون، مصمم ليعمل محلياً (Offline-first) كتطبيق ويب تقدمي (PWA) لضمان السرعة والأمان والوصول الدائم للبيانات بدون الحاجة لاتصال مستمر بالإنترنت.

## المميزات الرئيسية

*   **لوحة تحكم شاملة (Dashboard):** عرض إحصائيات حقيقية ومباشرة لجميع أقسام النظام مع رسوم بيانية توضيحية.
*   **متوافق مع جميع الأجهزة (Responsive):** تصميم مرن يتكيف تلقائياً مع الهواتف الذكية، الأجهزة اللوحية، وأجهزة الحاسوب.
*   **تصدير متقدم للبيانات:** إمكانية تحميل البيانات والتقارير بصيغ متعددة (Excel, PDF, Word) بضغطة زر واحدة.
*   **حاسبة متقدمة:** حاسبة مدمجة تدعم العمليات الحسابية المعقدة والمعادلات الرياضية.
*   **مساحة عمل ذكية:** محرر نصوص متقدم (يدعم القص، اللصق، التنسيق، الدمج) لكتابة الملاحظات، المهام، ومسودات العقود.
*   **إدارة متكاملة للأقسام:**
    *   المبيعات والمشتريات
    *   إدارة المخزون (مع دعم الباركود)
    *   العملاء والموردين (CRM)
    *   الموارد البشرية (HR)
    *   الاستيراد والتصدير
    *   المرتجعات والإلغاء
    *   الحسابات (إيرادات ومصروفات)
    *   إدارة الهالك والتوالف
    *   إصدار الفواتير
    *   عروض الأسعار وطلبات التسعير
    *   سجلات المعاملات (Log) للتدقيق والمراجعة

## التقنيات المستخدمة

*   **React & TypeScript:** لبناء واجهة مستخدم قوية وسريعة.
*   **Tailwind CSS:** لتصميم عصري ومتجاوب مع كافة الشاشات.
*   **Dexie.js (IndexedDB):** لقاعدة بيانات محلية سريعة وآمنة تعمل بدون إنترنت.
*   **Recharts:** للرسوم البيانية التفاعلية.
*   **Math.js:** لمعالجة المعادلات الرياضية المعقدة.
*   **Vite PWA:** لدعم تثبيت التطبيق على الأجهزة والعمل دون اتصال.
